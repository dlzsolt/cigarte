A simple Progressive Web App (PWA) to help reduce smoking by spacing cigarettes.

## Features
- Daily routine overview.
- Spacing plan (1 cigarette every 2 hours).
- Tracker with timer and daily count.
- Works offline (PWA).

## Deployment

### Vercel
1. Install Vercel CLI: `npm i -g vercel` (or use the web dashboard).
2. Run `vercel` in this directory.
3. Follow the prompts (keep defaults).

### GitHub Pages
1. Push this code to a GitHub repository.
2. Go to **Settings** > **Pages**.
3. Select the `main` branch and `/` root folder.
4. Save.

## Development
- Open `index.html` in your browser.
- To test PWA features (Service Worker), serve the folder using a local server (e.g., `npx serve .` or Live Server extension).
